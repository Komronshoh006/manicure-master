# Manicure Master — Landing Page Template

Небольшой шаблон сайта для мастера маникюра.  
Подойдёт для портфолио, рекламы услуг или личной страницы.

## Features
- Минималистичный дизайн  
- Блоки цен  
- Контактная форма  
- Адаптивная верстка  
- Быстрая загрузка

## Tech stack
- HTML  
- CSS  
- Без фреймворков (подходит новичкам)
